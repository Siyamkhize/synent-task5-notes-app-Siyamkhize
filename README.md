# Notes & Web Scraper App
**Developer: Siyabonga Mkhize**

A full-stack, cloud-native application for secure note-taking and e-commerce price tracking.

## 🚀 Live Demo
- **URL**: [https://notes-app-siya.onrender.com/](https://notes-app-siya.onrender.com/)
- **Tech Stack**: Python, Flask, SQLAlchemy, PostgreSQL, Google OAuth 2.0, BeautifulSoup4.

## ✨ Key Features
- **Secure Google Login**: Integrated with OAuth 2.0 for safe, password-less authentication.
- **Smart Scraper**: Automatically extracts product names and prices from any e-commerce URL.
- **Cloud Persistence**: Uses a managed PostgreSQL database on Render for high reliability.
- **Responsive Dashboard**: Clean, modern UI for managing notes and tracking prices.

## 🛠️ Technical Implementation
- **Backend**: Flask-based RESTful architecture.
- **Database**: SQLAlchemy ORM with PostgreSQL on Render.
- **CI/CD**: Automated deployment via GitHub and Render.
- **Security**: Environment variables for sensitive secrets and CSRF protection.
